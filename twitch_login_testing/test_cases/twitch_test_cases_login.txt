Название: Тест-кейсы для формы логина Twitch
Объект: https://www.twitch.tv/login
Версия: v1.0
Автор: Aleksandr Kuvychkin

TC_01 Позитивный сценарий: Успешный логин с валидными данными
Предусловие: Пользователь зарегистрирован
Шаги:
- Перейти на https://www.twitch.tv/login
- Ввести валидный "Username"
- Ввести валидный "Password"
- Нажать кнопку "Log In"
Ожидаемый результат:
- Пользователь успешно авторизован и перенаправлен на главную страницу

TC_02 Негативный сценарий: Пустые поля логина
Шаги:
- Перейти на https://www.twitch.tv/login
- Оставить оба поля пустыми
- Нажать кнопку "Log In"
Ожидаемый результат:
- Кнопка остаётся неактивной или появляется сообщение о пустых полях

TC_03 Негативный сценарий: Неверный пароль
Предусловие: Пользователь зарегистрирован
Шаги:
- Перейти на https://www.twitch.tv/login
- Ввести валидный "Username"
- Ввести неверный "Password"
- Нажать кнопку "Log In"
Ожидаемый результат:
- Отображается сообщение о неправильном пароле, вход не выполняется

TC_04 Негативный сценарий: Невалидный username
Шаги:
- Перейти на https://www.twitch.tv/login
- Ввести невалидный "Username"
- Ввести любой пароль
- Нажать кнопку "Log In"
Ожидаемый результат:
- Отображается сообщение о неправильном username или ошибке валидации

TC_05 Проверка ссылки "Trouble logging in?"
Шаги:
- Перейти на https://www.twitch.tv/login
- Кликнуть на "Trouble logging in?"
Ожидаемый результат:
- Открывается страница восстановления доступа
